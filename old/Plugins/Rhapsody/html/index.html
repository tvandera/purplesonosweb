This plugin allows you to interact with the Rhapsody service inside the Web Controller system. The plugin is needed because the Sonos controller software is what interacts with the Rhapsody services, and not the Sonos boxes themselves.

Known Issues

  • Enabling Rhapsody Search support in the web controller will slow down searches, since Rhapsody are slow and not local.
  • Enabling Rhapsody currently breaks iSonos
  • Web controller server must be able to reach the internet
  • Can not add/play a Rhapsody Artist

> Enable Rhapsody browse support
> Enable Rhapsody search support
> Enable My Rhapsody support - This will cause your Rhapsody password to be sent in the clear to Rhapsody
