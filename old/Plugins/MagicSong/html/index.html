This plugin allows any web page to be called when a "fake song" is added to the Sonos Queue. The plugin constantly monitors the Queue, and will automatically remove the song from the queue as it performs the action. For example a fake song could be added to automatically set up links.

The plugin requires you to configure a LOCAL directory that is indexed by the Sonos system. For security reasons you must do this by running "sonos.pl -config" from the command line.
