This plugin allows you to interact with your Sonos Zones by sending instant messages from AIM. It will login an AIM user that represents the Sonos Zones. (A screen name just for this purpose works best.) Then configured names can send ims to the Sonos screen name to control the systems.
  • Add names to the Allowed AIM Names list. These are the users who can control the AIM bot and send commands. All other ims and users will be logged and ignored.
  • Enter the AIM name and password for the Bot
  • Check the enable checkbox
  • Enable the zones that can be controlled
  • Remember to hit the Save button.
