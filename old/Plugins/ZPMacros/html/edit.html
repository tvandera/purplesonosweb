Fetch the following URL when the following sequence of buttons are hit in a short period of time. The URL can be anything, including external items.
Buttons:
M = Mute, U = Vol Up, D = Vol Down
Command:
URL:
%zone%, %artist%, %album%, %song%will be replaced in the urls.
So for example use %zone% for the current zone. Instead of using %zone% you can also force the action on one zone by using one of the RICON_* values below.
