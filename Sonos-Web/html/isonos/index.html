iSONOS! is an AJAX powered UI to the excellent Sonos Web Controller (www.purple.org/sonos/) and it has been heavily influenced by iTunes. iSONOS! uses some standard widgets found in OS X and iTunes however it is in no way affiliated with either Apple or Sonos, it's just a product of me wanting to learn AJAX and really digging both the Sonos music system and the Apple aesthetic. Please send any questions, comments or suggestions to me at isonos@yudev.com.
