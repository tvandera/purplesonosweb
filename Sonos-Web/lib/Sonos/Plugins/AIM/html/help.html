To save typing the Bot will try and guess both commands and zones from short hand.
For example, a zone named "Kitchen" can be refered to using just "kit" or "kitch".
Or the command "zone" can be shortened to just "z"

These commands require a zone to be selected first
CommandSyntaxSample
Zonezone {zone}z kitch
